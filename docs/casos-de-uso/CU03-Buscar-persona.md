# CU03 - Buscar persona

Actor: Usuario registrado

## Guión

1. Actor ingresa nombre
2. Sistema busca usuarios con ese nombre
3. Sistema muestra el login y nombre de los usuarios 

## Excepciones

**Excepción:** No se encuentran usuarios con ese nombre

2.1. Sistema muestra un mensaje "No se encuentran usuarios con ese nombre"
2.2. Termina
